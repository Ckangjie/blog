    # computed属性

    这是模板内容，在input绑定一个v-model，提供参考 ：

    声明一个计算属性computed，将changeValue 绑定到input上，其中getValue()为axios 请求方法。里面接收来自input框的值 也就是我们绑定的v-model=“input”，值为this.input。 

    computed: {
      changeValue: function(e) {
        // let value = this.input
        this.getValue(this.input)
        return this.input
      }
  }
